# Instruções para Codex — LPV Turismo

## Regras

- Não alterar URLs de imagens sem autorização.
- Não alterar páginas PT quando a tarefa for ES.
- Não alterar páginas ES quando a tarefa for PT.
- Não usar CSS genérico se puder usar `body.page-id-XX` ou uma classe específica.
- Validar tags HTML abertas/fechadas.
- Preservar as classes existentes do site: `topbar`, `container`, `hero`, `cards`, `lpv-service-card`, `lpv-simple-page`, `lpv-page-hero`, `lpv-content-section`, `process`, `cta`, `footer`.
- Formulários devem usar `https://formsubmit.co/info@lpvturismo.com`.
- Não usar `mailto:` para envio de formulário.

## Tom

PT: elegante, cultural, turístico, acolhedor.
ES: espanhol natural para argentinos/latinos, sem exagerar informalidade.
EN: inglês internacional, claro e comercial.
